## 感谢

- **Fluent Design 2 风格**：采用 [windows-ui](https://github.com/virtualvivek/windows-ui) 库，使用Html、CSS和JavaScript构建Windows原生外观和感觉的应用程序。
- **Markdown 优先**：集成 [markz](https://codeberg.org/sicher/markz) 解析器，一个用于 Zig 语言的高速 CommonMark Markdown 解析器。它构建完整的抽象语法树（AST）以支持程序化访问，并可选支持 GFM 表格/任务列表、CriticMarkup 以及 Obsidian 风格 Markdown 的扩展功能。
- **静态托管**：基于 [Ziex](https://github.com/ziex-dev/ziex) 一个用于Zig的全栈Web框架。使用熟悉的JSX模式编写声明式UI组件，编译为高效的Zig代码。
